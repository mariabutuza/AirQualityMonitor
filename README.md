Air Quality Monitor is a real-time air quality monitoring system. It uses an ESP8266 + MQ-135 sensor to detect pollutant levels and send the data to a microservices backend. Users can visualize measurements in real time via a React/Next.js web platform or an Android mobile app (Expo/React Native), and receive alerts when pollutant levels exceed predefined safety thresholds. The system continuously compares measured values against configurable pollution limits and automatically triggers notifications to warn users of dangerous air quality conditions. It uses JWT authentication and PostgreSQL for data storage.
